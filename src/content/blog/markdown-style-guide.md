Here is a sample of basic Markdown syntax supported in Astro.

## Headings

# H1
## H2
### H3

## Paragraph

Regular paragraph text. **Bold** and _italic_ are supported.

## Blockquotes

> Don't communicate by sharing memory, share memory by communicating.
> — Rob Pike

## Code Blocks

```js
const greeting = 'hello world';
console.log(greeting);
```

## Lists

- Item one
- Item two
- Item three

1. First
2. Second
3. Third

## Tables

| Name  | Value |
| ----- | ----- |
| Foo   | Bar   |
| Baz   | Qux   |
